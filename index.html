About

I am a PhD candidate in Geotechnical Engineering at Hassiba Benbouali University, Chlef, specializing in the development of interpretable machine learning frameworks for predicting soil behavior. My research focuses on integrating advanced gradient boosting algorithms (XGBoost, CatBoost, LightGBM) with explainable artificial intelligence methods (SHAP) to model complex soil-property relationships transparently and accurately.

My doctoral research addresses critical challenges in geotechnical engineering by developing data-driven models that bridge classical soil mechanics with modern computational intelligence. The work emphasizes both predictive accuracy and model interpretability, essential for engineering applications requiring transparent decision-making.

Research Projects

1. Compression Index Prediction with Interpretability

Developed a comprehensive framework for predicting compression index (Cc) of clayey soils using 1,243 soil samples. Implemented three gradient boosting algorithms (XGBoost, CatBoost, LightGBM) with Optuna-based hyperparameter optimization. Integrated SHAP analysis to provide transparent explanations of model predictions, revealing void ratio and water content as primary drivers of soil compressibility.

  • Best model performance: XGBoost (R² = 0.913, RMSE = 0.197, MAE = 0.100)
  • SHAP analysis: Mean SHAP values identify e₀ = 0.20, w = 0.10 as dominant features
  • Cross-validation: 5-fold cross-validation with rigorous train-test separation

2. Swelling Potential Prediction with Hybrid Metaheuristic Optimization

Developed PSO-XGBoost hybrid framework optimizing hyperparameters through particle swarm optimization across 81 scenarios. Conducted comprehensive global sensitivity analysis using Sobol indices and polynomial chaos expansion to quantify feature contributions and interactions.

  • Dataset: 292 soil samples from 10 literature sources with diverse swelling properties
  • PSO optimization: 100 iterations yielding superior generalization vs. baseline XGBoost
  • Global sensitivity: Initial water content identified as dominant factor (Sobol index = 0.855)

3. Swelling Pressure Prediction Using Multiple ML Algorithms

Comparative analysis of polynomial lasso regression, support vector regression, random forest, and XGBoost for predicting maximum swelling pressure. Database compilation of 266 samples after outlier removal from 5 published studies.

  • Best performer: XGBoost (R² = 0.934, MAE = 0.148 MPa, RMSE = 0.189 MPa)
  • Input features: PL, PI, γd, and initial water content
  • Superior accuracy over traditional empirical correlations

4. Handling Missing Data in Geotechnical Datasets

Systematic evaluation of three missing-value handling strategies: native algorithm support, median imputation, and K-nearest neighbors imputation. Dataset with 13 input features and up to 59% missing values (395 samples).

  • Finding: Native handling consistently outperformed explicit imputation methods
  • CatBoost native support: R² = 0.98 (RMSE = 3.39, MAE = 2.51)
  • Practical implication: Reduces preprocessing burden while improving accuracy

Tools & Web-Based Platform

Interactive Soil Swelling Potential Prediction Platform

Developed a comprehensive web application for practical deployment of machine learning models in geotechnical engineering. The platform integrates predictive modeling, interpretability analysis, and risk classification.

Access Web Platform

Platform Features

Single Sample Prediction

Interactive form for entering 13 soil properties. Returns predicted swelling potential with confidence intervals and risk classification (low/medium/high).

Batch Processing

Upload Excel files with multiple soil samples. Automatic processing with results including predictions, confidence intervals, SHAP contributions, and risk classifications for each case.

SHAP Interpretability

Interactive visualizations (waterfall plots, force plots, beeswarm summaries) explaining how each soil property contributes to predictions at both global and local levels.

Risk Classification System

Automated mapping of swelling potential to standard geotechnical risk categories. Provides distribution summaries for batch assessments to identify problematic soils.

Technical Architecture

The platform is built on a modern, scalable architecture combining frontend and backend components:

  • Frontend: Streamlit for interactive, browser-accessible user interface
  • ML Models: XGBoost, CatBoost, LightGBM with Optuna hyperparameter optimization
  • Interpretability: SHAP integration for both global and local explanations
  • Data Processing: Scikit-learn, Pandas, NumPy for preprocessing and transformation
  • Visualization: Matplotlib for high-quality SHAP visualizations
  • Missing Data: Support for median imputation, KNN imputation, and native algorithm handling

Engineering Applications

The platform provides practical benefits for geotechnical professionals:

  • Rapid preliminary estimates of soil swelling during early design phases
  • Transparent decision-making through SHAP-based feature explanations
  • Cost-effective alternative to time-intensive laboratory testing
  • Support for large-scale geotechnical site assessments through batch processing
  • Risk-informed design through standardized classification system
