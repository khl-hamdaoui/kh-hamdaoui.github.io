About Geotechnical Engineering Researcher

I am a PhD researcher in Geotechnical Engineering at Hassiba Benbouali University, Chlef, specializing in machine learning and deep learning applications for soil behavior prediction. My research focuses on developing interpretable predictive frameworks that integrate advanced gradient boosting algorithms with explainable artificial intelligence to model complex soil-property relationships accurately and transparently.

My work addresses critical challenges in geotechnical engineering by bridging classical soil mechanics with modern computational intelligence. The research emphasizes both predictive accuracy and model interpretability, essential for engineering applications requiring transparent, justified decision-making in civil engineering projects.

Geotechnical Engineering Research Projects

Machine Learning for Compression Index Prediction

Developed a comprehensive framework for predicting compression index of clayey soils using 1,243 soil samples and advanced machine learning. Implemented XGBoost, CatBoost, and LightGBM with Optuna hyperparameter optimization. Integrated SHAP for transparent soil behavior predictions.

  • Best performance: XGBoost (R² = 0.913, RMSE = 0.197, MAE = 0.100)
  • SHAP analysis: void ratio (0.20) and water content (0.10) as primary soil drivers
  • Validation: 5-fold cross-validation with rigorous train-test separation

Hybrid Metaheuristic Optimization for Swelling Potential

Developed PSO-XGBoost hybrid framework for swelling potential prediction with global sensitivity analysis. Evaluated 81 PSO scenarios using Sobol indices and polynomial chaos expansion for geotechnical soil characterization.

  • Dataset: 292 soil samples from 10 literature sources with diverse swelling properties
  • PSO optimization: 100 iterations showing superior machine learning generalization
  • Global sensitivity: Initial water content (Sobol index = 0.855) as dominant factor

Swelling Pressure Prediction Using Deep Learning Algorithms

Comparative analysis of polynomial lasso regression, support vector regression, random forest, and XGBoost for predicting maximum swelling pressure in expansive soils on 266 samples.

  • Best model: XGBoost (R² = 0.934, MAE = 0.148 MPa, RMSE = 0.189 MPa)
  • Input features: plastic limit, plasticity index, unit weight, initial water content
  • Performance: Superior accuracy compared to traditional geotechnical correlations

Handling Missing Data in Geotechnical Machine Learning

Systematic evaluation of three missing-value handling strategies on geotechnical dataset with 13 features and up to 59% missing values (395 soil samples).

  • Finding: Native algorithm handling outperformed explicit imputation methods
  • CatBoost native support: R² = 0.98 (RMSE = 3.39, MAE = 2.51)
  • Implication: Reduced preprocessing effort while improving prediction accuracy
