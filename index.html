About

I am a PhD researcher in Geotechnical Engineering at Hassiba Benbouali University, Chlef, specializing in machine learning and deep learning applications for soil behavior prediction. My research focuses on developing interpretable predictive frameworks that integrate advanced gradient boosting algorithms with explainable artificial intelligence to model complex soil-property relationships accurately and transparently.

My work addresses critical challenges in geotechnical engineering by bridging classical soil mechanics with modern computational intelligence. The research emphasizes both predictive accuracy and model interpretability, essential for engineering applications requiring transparent, justified decision-making.

Research Projects

Compression Index Prediction with Interpretability

Developed a framework for predicting compression index of clayey soils using 1,243 samples. Implemented XGBoost, CatBoost, and LightGBM with Optuna hyperparameter optimization. Integrated SHAP for transparent predictions.

  • Best performance: XGBoost (R² = 0.913, RMSE = 0.197, MAE = 0.100)
  • SHAP analysis: void ratio (0.20) and water content (0.10) as primary drivers
  • Validation: 5-fold cross-validation with rigorous train-test separation

Swelling Potential Prediction with Hybrid Metaheuristic Optimization

Developed PSO-XGBoost hybrid framework with global sensitivity analysis. Evaluated 81 PSO scenarios using Sobol indices and polynomial chaos expansion.

  • Dataset: 292 soil samples from 10 literature sources
  • PSO optimization: 100 iterations showing superior generalization
  • Global sensitivity: Initial water content (Sobol index = 0.855) as dominant factor

Swelling Pressure Prediction Using Multiple ML Algorithms

Comparative analysis of polynomial lasso regression, support vector regression, random forest, and XGBoost for swelling pressure prediction on 266 samples.

  • Best model: XGBoost (R² = 0.934, MAE = 0.148 MPa)
  • Features: PL, PI, γd, initial water content
  • Performance: Superior to traditional empirical correlations

Handling Missing Data in Geotechnical Datasets

Systematic evaluation of three missing-value handling strategies on dataset with 13 features and up to 59% missing values (395 samples).

  • Finding: Native handling outperformed explicit imputation
  • CatBoost native support: R² = 0.98 (RMSE = 3.39, MAE = 2.51)
  • Implication: Reduced preprocessing while improving accuracy

Tools and Web Platform

Interactive Soil Swelling Potential Prediction Platform

Comprehensive web application for deploying machine learning models in geotechnical engineering. Integrates predictive modeling, interpretability analysis, and risk classification.

Access Platform

Key Features

Single Prediction

Interactive form for 13 soil properties. Returns swelling potential with confidence intervals and risk classification.

Batch Processing

Upload Excel files with multiple samples. Automatic processing with predictions, confidence intervals, SHAP values, and risk classification.

SHAP Interpretability

Interactive visualizations explaining how soil properties contribute to predictions at global and local levels.

Risk Classification

Automated mapping to standard geotechnical categories. Distribution summaries for batch assessments.

Technical Architecture

  • Frontend: Streamlit interactive interface
  • ML Models: XGBoost, CatBoost, LightGBM with Optuna optimization
  • Interpretability: SHAP for global and local explanations
  • Data Processing: Scikit-learn, Pandas, NumPy
  • Visualization: Matplotlib for SHAP plots
  • Missing Data: Median, KNN imputation, and native handling

Engineering Applications
